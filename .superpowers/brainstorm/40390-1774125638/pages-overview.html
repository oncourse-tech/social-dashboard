Dashboard Pages — Full Feature Map

All pages mapped from the reference tool, adapted for oncourse

COMPETITOR INTEL
📊 Apps Overview
Summary cards (total apps, accounts, videos, viral counts).
Table: app name, # accounts, followers, likes, videos, 7d activity, 5K+, 50K+ counts.
Sortable columns. Add/delete apps.
👤 Accounts
Per-app filtered view. Summary cards (# accounts, total followers, total videos).
Table: username, app, last posted, followers, likes, videos, 7d, viral tiers.
+ Add Account button, search, filter by app.
👤 Account Detail
Profile header (avatar, username, app badge, tracking since, "View on TikTok" link).
Stats: followers, total likes, total videos, viral videos count.
Videos section: grid view (thumbnails) + list view (table) toggle.
Each video: thumbnail, description, views, likes, comments, shares, date, format label, viral tier badge.
Filter: All Videos / Viral Only / by format. Search by description/hashtag.
🔍 Account Research
Search TikTok for new accounts to track. Preview profile before adding.
Powered by Apify search or manual username entry.
📈 Posting Activity
Timeline/heatmap of when tracked accounts post.
Spot posting frequency patterns across competitors.
🔥 Viral Videos
Filtered view: only videos above configurable viral thresholds (default 5K+, 50K+).
Sortable by views, recency, growth rate. Filter by app, format, date range.
📋 All Videos
Unfiltered video browser across all tracked accounts.
Full search, sort, filter capabilities. Grid + list views.
📥 Bulk Import
CSV upload or paste list of usernames to bulk-add accounts.
Assign app + trigger initial sync.
CONTENT HOUSE
📊 Dashboard
Overview of your own oncourse TikTok performance.
Track your account's growth, recent posts, engagement trends.
💾 Saved Formats
Bookmark viral videos from competitors as "format templates".
Add notes on what makes it work, tags, reference links.
Replaces "Viral Carousels" from reference — covers all formats.
🎣 Hook Library
Collection of proven hooks (text openers, visual hooks, sound hooks).
Tag by format, source video, effectiveness notes.
🎬 Compositions
Draft content ideas by combining a format + hook + topic.
Status tracking: idea → scripted → filmed → posted.
📅 Content Calendar
Visual calendar for planned and published content.
Drag-and-drop scheduling, status colors, link to compositions.
👥 Our Accounts
Track oncourse's own TikTok accounts.
Same detail view as competitor accounts but marked as "ours".
🤖 AI Insights (Future)
Gemini-powered trend analysis, format recommendations,
auto-generated content briefs based on what's working.
