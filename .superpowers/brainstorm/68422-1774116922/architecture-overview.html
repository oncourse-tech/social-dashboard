Key Decisions

Data Source

Apify TikTok scrapers (~$30-60/mo), daily cron sync, webhook-based ingestion

Database

PostgreSQL via Supabase or Neon (free tier), Prisma ORM

Deploy

Vercel (Pro plan for 60s API routes), Vercel Cron for daily sync
